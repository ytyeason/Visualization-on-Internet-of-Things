# Nav Bar Component
A component integrated with react-router-dom component to achieve routing features

## Usage
import json file syntax :
```js
routes = [
    {
        exact: true,
        dispLabel: 'Menu 1',
        route: '/menu-1'
    },
    {
        dispLabel: 'Menu 2',
        route: '/menu-2'
    },
    {
        dispLabel: 'Menu 3',
        route: '/menu-3'
    },
]

```
